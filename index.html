Simple Obstacle Avoidance Algorithm for Autonomous Robot

Main idea

The robot has to find a path to escape from the maze.

How it works

A simple obstacle avoidance algorithm runs on the Arduino board. This algorithm allows the robot to explore its surrounding environment while this latter spins at its axis to find obstacles on four of its sides since the algorithm treats the robot as if it has a square form. The algorithm decides which movement to take based on the distance data from the robot to current obstacles obtained by the ultrasonic sensors attached to the robot.

The ultrasonic sensor detects the corner of the book, making the algorithm believe that there is an obstacle on the robot's front side; thus, the robot looks for another direction to move on.
